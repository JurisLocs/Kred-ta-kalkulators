# Kredīta Kalkulators

Vienkāršs kredīta kalkulators, kas ļauj aprēķināt aptuveno ikmēneša maksājumu atkarībā no kredīta summas un termiņa.

# Funkcijas

Kredīta summas un termiņa izvēle ar slīdņiem

Automātiska ikmēneša maksājuma aprēķināšana

Tumšais/gaišais režīms

# Lietošana

Izvēlies kredīta summu un termiņu, pārbīdot slīdņus.

Skaties, kā mainās ikmēneša maksājums.

Ja viss apmierina, nospied "Saņemt".

# Tehnoloģijas

HTML, CSS, JavaScript – vienkārša un efektīva darbība bez backend

Font Awesome – ikonas

# Instalācija

Nav nepieciešama instalācija – vienkārši atver index.html failu pārlūkā.
